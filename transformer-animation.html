    Key Concepts

    Self-Attention Formula

    Attention(Q,K,V) = softmax(QKT / √dk) · V

    Each token asks every other token: "How relevant are you to me?" Scores are normalized, then used to blend value vectors.

    Parallelization

    Unlike RNNs, all tokens are processed simultaneously. Self-attention computes all pairwise relationships in one matrix operation — enabling massive GPU parallelism.

    Context Window

    The maximum sequence length the model can attend over. Larger windows (e.g. 128K tokens) let the model consider more context, but increase memory quadratically with attention.

    Example Sentence

    "i like data science and machine learning"

    7 tokens — a perfect length for tracing attention patterns across two related phrases.
